# Maxxidata Challenge

Tecnologias utilizadas no frontend:
  - ReactJS
  - MaterialUI
  - React Router
  - Yup validator

Tecnologias utilizadas no backend:
  - NodeJS
  - Postgres
  - TypeScript
  - ExpressJS
  - TypeORM
  - Yup validator


## Backend

Para rodar o backend com o docker basta:
```sh
docker-compose up --build
```

Para rodar com o npm basta:
```sh
npm install
npm run start
```

## Frontend

Para rodar o frontend basta:
```sh
npm install
npm run start
```
